Multiplayer Tic Tac Toe

Source Code

Multiplayer TicTacToe game where player can play games with other players or computer AI. Computer AI is unbeatable using minimax algorithm. Player can also spectate other game happening between other players.

Player can log in and see lifetime Win/Lose/Draw records and the history of each game (winner, list of moves, viewers of game).

Concepts applied: OOP, concurrency, socket, minimax algorithm

Technologies used: Java, JavaFX

Multiplayer Tic Tac Toe - Server

Source Code

Server part of the Multiplayer TicTacToe game. Server can monitor all active players and all games playing. Server can also drill down to specific player and game detail.

Server can access the database of all users and all previous games played.

Concepts applied: OOP, concurrency, socket

Technologies used: Java, JavaFX, SQLite
